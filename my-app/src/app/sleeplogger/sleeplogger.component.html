Ionic 4: Sleep logger app







Introduction

This app is tool for indivivduals to track sleep and sleepiness in the Ionic framework.

Tasks that users can do is

1. The ability to log sleepiness during the day.

2. The ability to view these two categories of logged data

3. The ability to view these two categories of logged data.

Logging Overnight Sleep

The way in which we implemented how to log a person's overnight sleep cycle was incremental. The user starts the sleep log, and in the morning the user ends the sleep log. This feature was most prevalent in other popular sleeptracking application in the market place and we felt it was a useful way of capturing the amount of time a user spent sleeping.



Logging Sleepiness

Sleepiness is logged by selecting the specific tab at the bottom of the application. The user will be prompted to select the specific level of sleepiness they are experiencing. Since it is possible for the user to feel varying levels of sleepiness during the day, the application supports multiple entries. To track each entry the application notifies the user after each entry. In addition, the application will give consistent reminders of when to log by a local notification that pops up at 12pm, 4pom and 9pm every day. That way, the user does not forget when to log their sleepiness data.



Viewing their data

The application supports data viewing through the data tab in the application located in the bottom portion of the application. The data is displayed in interactive charts that offer the user more detailed information when they select specific data entries. The data can be toggled between how sleep duration data and sleepiness data. In addition, users can see their recent entries that they made as well.
